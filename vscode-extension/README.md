# AI Agent Platform — VS Code Extension

Send requests to AI Agent Platform directly from VS Code.

## Commands

- **AI Agent: Send Request** (`Ctrl+Shift+A`) — send selected code or query
- **AI Agent: Apply Changes** — apply agent's suggested changes
- **AI Agent: Show History** — view request history
- **AI Agent: Set Project Path** — configure project

## Configuration

- `aiagent.apiUrl` — agent URL (default: http://localhost:5000)
